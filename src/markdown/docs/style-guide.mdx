# Cont, autentificare și securitate

În AInevoie poți folosi autentificarea prin:

- **Email și parolă**
- **Link magic** (unde este disponibil)
- **Google** (unde este disponibil)

## Recomandări pentru parolă

Pentru o securitate mai bună, folosește o parolă care include:

- cel puțin o literă mare
- cel puțin o literă mică
- cel puțin o cifră
- cel puțin un caracter special

## Resetarea parolei

Dacă ai uitat parola, poți solicita un link de resetare, apoi îți setezi o parolă nouă.

## Confidențialitate

Datele sunt folosite pentru funcționarea platformei (cont, solicitări, programări, plăți — unde e cazul) și pentru îmbunătățirea experienței în aplicație.
