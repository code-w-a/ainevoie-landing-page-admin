# Panou Admin (web)

Admin Web este aplicația de administrare a platformei AInevoie. Este optimizată pentru utilizare pe desktop și poate fi folosită și ca PWA (instalabilă din browser).

## Ce poți gestiona din Admin

- **Utilizatori**: listă, căutare, filtrare, activare/dezactivare conturi
- **Furnizori**: verificare date, aprobări, status, gestionare profiluri
- **Servicii și categorii**: adăugare / editare / ștergere
- **Cereri și programări**: vizualizare, status, monitorizare solicitări
- **Feedback / evaluări**: vizualizare și moderare (dacă este necesar)
- **Statistici de bază**: utilizatori, furnizori activi, solicitări, programări
- **Export date**: acolo unde este necesar (format tabelar)
- **Tarife/Prețuri**: setare și actualizare tarife pentru servicii/categorii (fără intervenție tehnică)

## Notă despre tarife

Modificările de tarife realizate din Admin se aplică solicitărilor viitoare și nu modifică retroactiv comenzile deja confirmate sau tranzacțiile efectuate.
